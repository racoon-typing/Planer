// Инструкция

1. Это приложения для хранения списка задач

2. Для того чтобы добавить задачу:

2.1 Заполните все поля. Они обязательные*
Заголовок: длина от 0 до 20 символов (Кириллица).
Текст: длина от 0 до 100 символов (Кириллица).

2.2 Нажмите кнопку отправить

2.3 Если вы указали дату, которая уже есть в списке, то задача добавится в эту группу.
В другом случае появится новая группа задач

3 Чтобы удалить задачу нажмите на нее, она будет подсвечена и появится крестик. Нажмите на крестик.
